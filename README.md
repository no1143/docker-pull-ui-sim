## English Introduction
This project is a Python-based terminal UI simulator that replicates the visual style of Docker image pulling.  
Features include:
- Braille-based progress bar (each character represents 8 sub-blocks)
- Yellow progress bar animation
- Rotating spinner symbols (⠋ ⠙ ⠸ ⠴ ⠦ ⠇)
- Display of current and total download size
- Green checkmark ✓ and "Done" status when completed

This project is just a prototype idea — a starting point for others to build upon.  
Feel free to modify, extend, or improve it. The code is licensed under MIT, so you can use it freely.

## 中文介绍
这是一个基于 Python 的终端界面模拟器，用来 **1:1 复刻 Docker 拉取镜像时的界面效果**。  
主要功能：
- 基于 Braille 的进度条（每个字符包含 8 个小方块）
- 黄色进度条动画
- 动态旋转符号（⠋ ⠙ ⠸ ⠴ ⠦ ⠇）
- 显示当前下载量和总下载量
- 下载完成后显示绿色对勾 ✓ 和 “Done” 状态

这个项目只是一个 **抛砖引玉** 的小想法 —— 给大家提供一个起点。  
欢迎随意修改、扩展或改进。代码已经采用 MIT 许可，可以自由使用。
